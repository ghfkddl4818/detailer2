📋 이커머스 상세페이지 컨설팅 업무 완전 가이드
🎯 비즈니스 모델 개요
핵심 아이디어: 네이버 스마트스토어 판매자들의 상세페이지를 분석해서 개선점을 찾아내고, 콜드메일로 영업한 후 맞춤형 기획안을 판매하는 B2B 컨설팅 서비스
수익 구조: 분석 → 콜드메일 영업 → 기획안 제작 → 판매
________________________________________
🔍 Phase 1: 영업 프로세스 (콜드메일 발송)
1-1. 타겟 고객 발굴
대상: 네이버 스마트스토어 운영자 선별 기준:
•	관심고객수: 50~1500명
•	리뷰 수: 300개 정도
•	제품군: 제한 없음 (모든 카테고리 대상)
작업 과정:
1.	네이버 스마트스토어에서 조건에 맞는 업체 검색
2.	엑셀 양식에 정보 입력: 
o	제품 URL
o	스토어 URL
o	스토어명, 제품명
o	관심고객수, 리뷰수
o	진행 상태
1-2. 데이터 수집
수집 도구:
•	FireShot: 상세페이지 전체 캡처 (JPG)
•	이커머스AI솔루션: 리뷰 데이터 크롤링 (엑셀 → TXT 변환)
수집 항목:
•	상세페이지 이미지 (전체 스크린샷)
•	고객 리뷰 데이터 (평점, 내용, 날짜 등)
•	기본 상품 정보 (가격, 스펙 등)
1-3. 콜드메일 생성 및 발송
사용 도구: 제미나이 2.5 Pro (구글 AI 스튜디오, 온도값 0.3)
프롬프트 구조:
•	SECTION A: 데이터 분석 보고서 (내부용)
•	SECTION B: 12단계 상세페이지 기획안 (고객 납품용)
•	TRUST 모델 기반 심리학적 분석
발송 방식: 구글메일 (개별 발송, 일괄발송 기능 원함)
하루 목표량: 50개 이상
________________________________________
💼 Phase 2: 기획안 제작 프로세스
2-1. 답변 고객 관리
답변 분류:
•	✅ 관심 있음 → 기획안 제작 대상
•	❌ 거절 → 거절 리스트 이동
•	⏳ 보류 → 팔로업 스케줄링
2-2. 기획안 제작 준비
클로드 사용 이유: JPG 파일 길이 제한으로 OCR 필요
OCR 작업:
•	제미나이에서 OCR 프롬프트 실행 (온도값 0.1)
•	상세페이지 이미지 → 텍스트 변환
필요 데이터:
•	고객사 OCR 결과
•	고객사 리뷰 데이터
•	경쟁사 OCR 데이터 (3개 업체)
•	경쟁사 리뷰 데이터 (3개 업체)
2-3. 기획안 생성
사용 도구: 클로드 (기획안 프롬프트)
결과물: 즉시 적용 가능한 12단계 상세페이지 기획안
•	컬러 팔레트 제안
•	단계별 헤드라인/본문/이미지 설명
•	전체 톤앤매너 가이드

🚨 현재 Pain Points
효율성 문제
1.	파일 정리: 다운로드 폴더에서 일일이 파일 찾아서 정리
2.	체험판 의존: FireShot, 이커머스AI 모두 유료 결제 필요
3.	수동 작업: 하나하나 다른 이름으로 저장하기 빡셈
기술적 제약
1.	JPG 길이 제한: 클로드에서 긴 상세페이지 처리 불가
2.	파일 변환: 엑셀 → TXT 수동 변환
3.	캡처 실패: 너무 긴 페이지는 캡처 도구들이 포기
확장성 한계
1.	일괄 발송: 구글메일 개별 발송만 가능
2.	자동화 부족: 대부분 수동 작업
3.	버전 관리: 프롬프트 업데이트 시 관리 복잡
